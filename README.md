# Platemate

**Pl8m8** — 식당 QR코드 하나로 주문·결제·리뷰까지

테이블에 놓인 QR코드를 손님이 스캔하면, 메뉴 보기·주문·결제가 한 번에 이루어집니다.  
외국인 손님도 자기 언어로 메뉴를 보고, AI가 메뉴를 설명해 주며, 편하게 결제할 수 있습니다.  
사장님은 대시보드에서 주문을 관리하고, 손님의 비공개 리뷰를 AI가 정리한 주간 리포트로 확인할 수 있습니다.

---

## 이 서비스로 할 수 있는 것

### 손님에게

- **테이블 QR만 스캔** — 전단지·종이 메뉴 없이 스마트폰으로 메뉴 확인
- **내 언어로 보기** — 한국어, 영어, 중국어, 일본어 중 선택해 메뉴와 설명 확인
- **AI 메뉴 설명** — 메뉴를 누르면 재료, 맛 특징, 어울리는 조합 등을 AI가 쉽게 설명
- **바로 주문·결제** — 장바구니에 담고, 한국 카드(토스페이먼츠) 또는 해외 결제(알리페이·위챗·카드 등)로 결제
- **비공개 리뷰** — 결제 후 음식·서비스 평가와 좋았던 메뉴를 남기면, 사장님만 볼 수 있는 리뷰로 저장

### 사장님(사업자)에게

- **한 곳에서 주문 관리** — 들어온 주문을 실시간으로 확인하고 상태(접수·조리 중·완료 등) 관리
- **메뉴·테이블 관리** — 메뉴 추가·수정·사진, 테이블별 QR코드 생성·다운로드
- **비공개 리뷰 요약** — 손님이 남긴 리뷰를 AI가 주간 단위로 분석해, 인기 메뉴·개선점·칭찬 내용을 한눈에 보는 리포트 제공
